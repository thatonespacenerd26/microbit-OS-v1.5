# microbit-OS-v1.5
a pretty cool entire OS for the micro:bit v2, it has 7 apps. a data logger, a simple security system, a tilt tracker, a volume tracker, a manual sleep mode, a dice roller, and a app where if you plug it into a breakout board which is plugged into a tm1637, the tm1637 starts counting from 1 to 9999. auto sleep mode after 20 secs of inactivity.enjoy!
