# microbit-OS-v1.5
a pretty cool entire OS for the micro:bit v2, it has 7 apps. a data logger, a simple security system, a tilt tracker, a volume tracker, a manual sleep mode, a dice roller, and a app where if you plug it into a breakout board which is plugged into a tm1637, the tm1637 starts counting from 1 to 9999. auto sleep mode after 20 secs of inactivity.
remember, this os is built for the micro:bit v2 only. it will receive updates occasionally, around every 1 and a half months. 
the OS has a little extra feature, you have to shake the micro:bit and on screen, you will see the cpu temperature of the micro:bit. WARNING:the temperature reading may not be perfectly accurate and may vary up to 5 celsius, because the micro:bit is not a scientific thermometer :)
have a great time using the micro:bit OS!
